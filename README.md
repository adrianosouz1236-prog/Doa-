**Plataforma web para conectar doadores a ONGs, promovendo solidariedade e inclusão digital**

## 📋 Sobre o Projeto

A **Plataforma Doa+** é um sistema web desenvolvido para conectar pessoas que desejam realizar doações a Organizações Não Governamentais (ONGs) que necessitam de recursos para suas atividades sociais.

O projeto nasceu como um portfólio pessoal de desenvolvimento full-stack e foi adaptado para atender aos objetivos da **Atividade Extensionista** do curso de **Análise e Desenvolvimento de Sistemas** do 
Centro Universitário Internacional UNINTER.

### 🌍 Público-alvo

- **ONGs** localizadas em Ananindeua/PA e região metropolitana
- **Doadores** da comunidade que desejam contribuir com causas sociais
- **Comunidades em situação de vulnerabilidade** atendidas pelas ONGs parceiras

## 🎯 Objetivos

### Objetivo Geral
Desenvolver e implementar uma plataforma tecnológica segura que conecte doadores da comunidade a Organizações Não Governamentais, facilitando o processo de doação e promovendo a cultura da solidariedade.

### Objetivos Específicos
1. Implementar um sistema web completo (frontend, backend e banco de dados) com autenticação segura
2. Desenvolver funcionalidades de busca inteligente por localização, categoria e urgência
3. Aplicar medidas de segurança cibernética (proteção contra SQL Injection, rate limiting, logs de auditoria)
4. Capacitar as equipes das ONGs parceiras no uso da plataforma
5. Avaliar o impacto social do projeto através de métricas quantitativas e qualitativas
